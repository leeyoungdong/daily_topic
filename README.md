# 목적
* CS 및 키워드공부는 암기가 아닌 응용이 중요 하다고 생각했다. 다양한 상황에서 접목시키고, 다양한 상황을 떠올리며 대응해야 한다고 생각했다. 그래서 간단하게 물음을 던지고, 답하는 식으로 한줄씩 늘려가볼까 한다.

# 방법

* 거창한 키워드가 아니여도 좋다. 간단한 키워드로써, 내가 CS, DATA, NETWORK등.. 다양하게 컴퓨터 관련이라면 하루에 하나씩 해보자
